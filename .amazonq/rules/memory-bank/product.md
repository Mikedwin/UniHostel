# UniHostel - Student Accommodation Marketplace

## Project Purpose
UniHostel is a dedicated full-stack web application that bridges the gap between university students searching for safe, affordable housing and hostel managers looking to fill vacancies. The platform creates a specialized marketplace focused exclusively on student accommodation needs.

## Value Proposition
- **Targeted Solution**: Purpose-built for the student housing market, addressing unique needs like semester-based bookings
- **Dual-User Experience**: Optimized workflows for both students seeking accommodation and managers listing properties
- **Trust & Safety**: Secure authentication and application tracking system builds confidence for both parties
- **Efficiency**: Streamlined booking process eliminates traditional housing search friction

## Key Features & Capabilities

### For Students
- **Smart Search & Discovery**: Browse hostels with advanced filtering by location, price range, and available facilities
- **Semester-Based Applications**: Apply for specific academic terms with personalized messages to managers
- **Application Tracking**: Real-time status monitoring (Pending/Approved/Rejected) through dedicated dashboard
- **Responsive Experience**: Mobile-optimized interface for on-the-go housing searches

### For Hostel Managers
- **Property Management**: Easy hostel listing with detailed property information and amenities
- **Application Processing**: Centralized dashboard to review, approve, or reject student applications
- **Availability Tracking**: Monitor occupancy and manage bookings across different time periods
- **Direct Communication**: Receive and respond to student inquiries through the platform

### Technical Capabilities
- **Secure Authentication**: JWT-based security with encrypted password storage
- **Real-time Updates**: Dynamic application status changes and instant notifications
- **Scalable Architecture**: MongoDB database with Express.js API for reliable performance
- **Modern UI/UX**: React-based frontend with Tailwind CSS for professional, responsive design

## Target Users

### Primary Users
- **University Students**: Seeking safe, affordable accommodation near campus for specific semesters
- **Hostel Managers/Landlords**: Property owners looking to efficiently manage student housing bookings

### Use Cases
- **Student Housing Search**: Students can quickly find and apply for accommodation that matches their budget and location preferences
- **Property Marketing**: Managers can showcase their hostels to a targeted student audience
- **Booking Management**: Streamlined application and approval process for both parties
- **Semester Planning**: Students can secure housing for upcoming academic terms in advance
